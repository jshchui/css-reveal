Other cool stuff you should know about

  • <search> -> container used to semantically represent parts of the document with search capabilities.
  • content-visiblility -> property used to skip an element's rendering work until needed.
  • ESLint -> CSS is now supported by ESLint for linting, offring additional rules for checking stylesheets.
  • :user-valid and :user-invalid psuedo-classes -> gives feedback only after a user has changed the input.
