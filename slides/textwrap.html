What about <white-space>?

text-wrap

  • Adjusts how text wraps when it reaches the end of a line
  • Does not affect how spaces or newlines are treatedâ€”only how words break.
  • Useful for improving text readability and layout optimization.

white-space

  • Determines whether text wraps at all (normal, nowrap).
  • Affects how spaces and newlines are preserved (pre, pre-wrap, pre-line).
  • Used for handling text formatting (e.g., preserving spacing in code blocks).
