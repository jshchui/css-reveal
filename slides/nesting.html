CSS Nesting

  • Previously, every CSS selector needed to be explicitly declared, separately from one another.
  • This added a lot of reptition and unnecssary complexity to your stylesheets.
  • As a workaround, developers turned to CSS extensions such as SASS/SCSS.
  • With CSS nesting, developers can now nest one style rule inside another, readability, modularity, and maintainability of stylesheets.
