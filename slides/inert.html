Inert

  • The inert property when present, makes the browser "ignore" user input events for the element.
  • This is great for accessibility as it removes focus events and elements from assistive technologies.
  • Previously, developers had to create a "focus trapping" strategy.
  • With inert, no trapping is required because you can freeze or guard entire sections of the page or app.
